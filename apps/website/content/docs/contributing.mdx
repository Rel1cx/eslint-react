## Packages Summary

This section provides a summary of the packages in the monorepo.

### Local Packages

- [`.pkgs/configs`](https://github.com/Rel1cx/eslint-react/tree/main/.pkgs/configs): Workspace config bases
- [`.pkgs/eff`](https://github.com/Rel1cx/eslint-react/tree/main/.pkgs/eff): JavaScript and TypeScript utilities

### Internal Packages

- **Utilities**
  - [`packages/ast`](https://github.com/Rel1cx/eslint-react/tree/main/packages/ast): TSESTree AST utility module for static analysis
  - [`packages/var`](https://github.com/Rel1cx/eslint-react/tree/main/packages/var): TSESTree AST utility module for static analysis of variables
  - [`packages/jsx`](https://github.com/Rel1cx/eslint-react/tree/main/packages/jsx): TSESTree JSX utility module for static analysis of JSX patterns
- **Core & Shared**
  - [`packages/core`](https://github.com/Rel1cx/eslint-react/tree/main/packages/core): Utility module for static analysis of React core APIs and patterns
  - [`packages/shared`](https://github.com/Rel1cx/eslint-react/tree/main/packages/shared): Shared constants, types and functions

### Public Packages

- **Utilities**
  - [`packages/kit`](https://github.com/Rel1cx/eslint-react/tree/main/packages/kit): Utility module for building custom React rules with JavaScript functions
- **ESLint Plugins**
  - [`plugins/eslint-plugin-react-x`](https://github.com/Rel1cx/eslint-react/tree/main/plugins/eslint-plugin-react-x): Core React rules
  - [`plugins/eslint-plugin-react-jsx`](https://github.com/Rel1cx/eslint-react/tree/main/plugins/eslint-plugin-react-jsx): React Flavored JSX rules
  - [`plugins/eslint-plugin-react-rsc`](https://github.com/Rel1cx/eslint-react/tree/main/plugins/eslint-plugin-react-rsc): React Server Components rules
  - [`plugins/eslint-plugin-react-dom`](https://github.com/Rel1cx/eslint-react/tree/main/plugins/eslint-plugin-react-dom): React DOM rules
  - [`plugins/eslint-plugin-react-web-api`](https://github.com/Rel1cx/eslint-react/tree/main/plugins/eslint-plugin-react-web-api): Web API interaction rules
  - [`plugins/eslint-plugin-react-naming-convention`](https://github.com/Rel1cx/eslint-react/tree/main/plugins/eslint-plugin-react-naming-convention): Naming convention rules
  - [`plugins/eslint-plugin-react-debug`](https://github.com/Rel1cx/eslint-react/tree/main/plugins/eslint-plugin-react-debug): Debugging rules for inspecting React patterns in code
  - [`plugins/eslint-plugin`](https://github.com/Rel1cx/eslint-react/tree/main/plugins/eslint-plugin) (npm: `@eslint-react/eslint-plugin`): A unified plugin that combines all individual plugins into one

### Documentation

- [`apps/website`](https://github.com/Rel1cx/eslint-react/tree/main/apps/website): Documentation website
- [`apps/playground`](https://github.com/Rel1cx/eslint-react/tree/main/apps/playground): Interactive playground (placeholder)
