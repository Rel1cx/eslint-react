JSX-specific utilities for analyzing JSX elements, attributes, and children.

## Installation

```sh
npm install --save-dev @eslint-react/jsx
```

## Overview

The `@eslint-react/jsx` package provides utilities for:

- **Element Analysis** - Identify and classify JSX elements
- **Attribute Handling** - Find, get, and validate JSX attributes
- **Children Inspection** - Work with JSX children
- **Value Resolution** - Resolve static and dynamic attribute values

## API Reference

### Element Identification

Functions for identifying JSX element types:

| Function                  | Description                                                     |
| ------------------------- | --------------------------------------------------------------- |
| `isElement(node)`         | Check if node is a JSX element                                  |
| `isFragmentElement(node)` | Check if node is a JSX fragment (`<>...</>`)                    |
| `isHostElement(node)`     | Check if node is a host/DOM element (lowercase tag)             |
| `isJsxLike(node)`         | Check if node looks like JSX (including custom implementations) |
| `isJsxText(node)`         | Check if node is JSX text content                               |

### Element Information

| Function               | Description                        |
| ---------------------- | ---------------------------------- |
| `getElementType(node)` | Get the type/name of a JSX element |
| `isWhitespace(node)`   | Check if node is whitespace text   |

### Attribute Handling

#### Finding Attributes

| Function                          | Description                                      |
| --------------------------------- | ------------------------------------------------ |
| `findAttribute(node, name)`       | Find a specific attribute by name                |
| `findParentAttribute(node, name)` | Find attribute in parent element                 |
| `hasAttribute(node, name)`        | Check if element has a specific attribute        |
| `hasAnyAttribute(node, names)`    | Check if element has any of the given attributes |
| `hasEveryAttribute(node, names)`  | Check if element has all given attributes        |

#### Getting Attribute Values

| Function                        | Description                              |
| ------------------------------- | ---------------------------------------- |
| `getAttributeName(node)`        | Get the name of an attribute             |
| `getAttributeValue(node)`       | Get the value of an attribute            |
| `getAttributeStaticValue(node)` | Get the static (evaluable) value         |
| `resolveAttributeValue(node)`   | Resolve the actual value of an attribute |

### Children Handling

| Function            | Description                       |
| ------------------- | --------------------------------- |
| `hasChildren(node)` | Check if element has children     |
| `getChildren(node)` | Get all children of a JSX element |

### JSX Value Types

Utilities for working with JSX attribute values:

| Function                     | Description                            |
| ---------------------------- | -------------------------------------- |
| `getJsxAttributeValue(node)` | Extract the value from a JSX attribute |
| `isJsxAttributeValue(node)`  | Check if node is a JSX attribute value |

### Configuration

| Constant           | Description                          |
| ------------------ | ------------------------------------ |
| `JSX_CONFIG`       | Default JSX configuration values     |
| `JsxDetectionHint` | Bit flags for JSX detection behavior |

## Usage Examples

### Finding an Attribute

```ts
import { findAttribute, getAttributeValue } from "@eslint-react/jsx";
import type { TSESTree } from "@typescript-eslint/utils";

function getSrcAttribute(node: TSESTree.JSXOpeningElement) {
  const srcAttr = findAttribute(node, "src");
  if (srcAttr) {
    return getAttributeValue(srcAttr);
  }
  return null;
}
```

### Checking for Dangerous HTML

```ts
import { hasAttribute, getAttributeValue } from "@eslint-react/jsx";

function hasDangerouslySetInnerHTML(node: TSESTree.JSXOpeningElement) {
  if (hasAttribute(node, "dangerouslySetInnerHTML")) {
    const attr = findAttribute(node, "dangerouslySetInnerHTML");
    const value = getAttributeValue(attr);
    // ... validate the value
    return true;
  }
  return false;
}
```

### Working with Children

```ts
import { getChildren, isWhitespace } from "@eslint-react/jsx";

function getNonWhitespaceChildren(node: TSESTree.JSXElement) {
  const children = getChildren(node);
  return children.filter(child => !isWhitespace(child));
}
```

### Resolving Static Values

```ts
import { getAttributeStaticValue } from "@eslint-react/jsx";

function getStaticHref(node: TSESTree.JSXOpeningElement) {
  const href = findAttribute(node, "href");
  if (href) {
    // Returns the actual value if it can be statically determined
    const staticValue = getAttributeStaticValue(href);
    return staticValue;
  }
  return undefined;
}
```
