ESLint React provides a set of utility modules designed to help you build custom lint rules for React projects. These utilities offer low-level AST manipulation, JSX analysis, and variable tracking capabilities.

<Callout type="warning">
  Modules marked as **Beta** may have API changes in future releases. Please test
  thoroughly before using in production.
</Callout>

<Callout type="error">
  Modules marked as **Unstable** may have significant API changes or be removed
  in future releases. Use with caution and expect breaking changes.
</Callout>

## Getting Started

If you're building custom rules, start with the [**Kit**](/docs/utilities/kit) documentation. For more specific use cases, refer to the individual module documentation:

- [**AST**](/docs/utilities/ast) - Working with AST nodes
- [**JSX**](/docs/utilities/jsx) - Analyzing JSX structures
- [**Var**](/docs/utilities/var) - Tracking variable assignments
