## Rule Details

Scheduling an interval within the setup function of `useEffect` without canceling it in the cleanup function can lead to unwanted `setInterval` callback executions and may also result in using stale values captured by previous renders' effects after each subsequent re-render.

## Examples

### Setting interval without cleanup in useEffect

```tsx
import React, { useEffect, useState } from "react";

function MyComponent() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    // Problem: A 'setInterval' created in 'useEffect' must be cleared in the cleanup function.
    const intervalId = setInterval(() => console.log(count), 1000);
  }, []);

  return null;
}
```

### Clearing interval in useEffect cleanup

```tsx
import React, { useEffect, useState } from "react";

function MyComponent() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    // Recommended: Store the interval ID
    const intervalId = setInterval(() => console.log(count), 1000);
    // Recommended: Clear the interval in the cleanup function
    return () => clearInterval(intervalId);
  }, []);

  return null;
}
```

### Avoiding stale dependencies in intervals

If your interval callback reads state, including that state in the dependency array will reset the interval on every change. Use a state updater function instead so the interval can run with an empty dependency array.

```tsx
// Recommended: use a state updater to avoid stale closures and unnecessary resets
import { useState, useEffect } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    const intervalId = setInterval(() => {
      setCount((c) => c + 1);
    }, 1000);
    return () => clearInterval(intervalId);
  }, []);

  return <h1>{count}</h1>;
}
```

### Encapsulating interval logic in a custom hook

When multiple components need the same interval behavior, extract it into a custom hook. This keeps the cleanup logic centralized and ensures every `setInterval` has a matching `clearInterval`.

```tsx
// Recommended: encapsulate interval logic in a custom hook with cleanup
import { useState, useEffect } from "react";

function useTimer(callback: () => void, delay: number) {
  useEffect(() => {
    const id = setInterval(() => {
      callback();
    }, delay);
    return () => {
      clearInterval(id);
    };
  }, [delay, callback]);
}

function Timer() {
  const [count, setCount] = useState(0);

  useTimer(() => {
    setCount((c) => c + 1);
  }, 1000);

  return <h1>Counter: {count}</h1>;
}
```
