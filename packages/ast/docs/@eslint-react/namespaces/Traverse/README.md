# Traverse

Helpers for traversing `TSESTree` nodes.

## Functions

| Function                                                    | Description                                                                                   |
| ----------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| [findEnclosingTryBlock](functions/findEnclosingTryBlock.md) | Finds the nearest TryStatement whose `try` block (not catch/finally) encloses the given node. |
| [findParent](functions/findParent.md)                       | Walk up the AST from `node` to find the nearest ancestor matching a predicate.                |
