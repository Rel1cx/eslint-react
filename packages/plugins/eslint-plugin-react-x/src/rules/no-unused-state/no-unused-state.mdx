## Rule Details

## Common Violations

### Invalid

```tsx
import React from "react";

class MyComponent extends React.Component {
  // Unused
  state = {
    foo: 1,
  };

  render() {
    return null;
  }
}
```

### Valid

```tsx
import React from "react";

class MyComponent extends React.Component {
  state = {
    foo: 1,
  };

  render() {
    return this.state.foo;
  }
}
```
