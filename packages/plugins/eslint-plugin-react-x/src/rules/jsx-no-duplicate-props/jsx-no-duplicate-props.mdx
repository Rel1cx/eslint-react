## Rule Details

Passing the same prop twice in a JSX element will cause the second value to overwrite the first. This is usually a mistake and can lead to unexpected behavior.

## Common Violations

### Invalid

```tsx
<Hello name="John" name="Doe" />;
```

### Valid

```tsx
<Hello name="John" />;
```
