## Description

Enforces that all contexts have a `displayName` that can be used in DevTools.

## Examples

### Failing

```tsx
import React from "react";

const MyContext = React.createContext();
```

### Passing

```tsx
import React from "react";

const MyContext = React.createContext();
MyContext.displayName = "MyContext";
```
