## Description

Prevents using variables in JSX that are not defined in the scope.

## Examples

### Failing

```tsx
import React from "react";

const button = <MyButton />;
//              ^^^^^^^^
//              - 'MyButton' is not defined
```

### Passing

```tsx
import React from "react";
import MyButton from "./MyButton";

const button = <MyButton />;
```
