---
title: no-string-style-prop
description: Disallows the use of string style prop in JSX. Use an object instead.
---

**Full Name in [`@eslint-react/eslint-plugin`](https://npmx.dev/package/@eslint-react/eslint-plugin/v/latest)**

```plain copy
@eslint-react/dom/no-string-style-prop
```

**Full Name in `eslint-plugin-react-dom`**

```plain copy
react-dom/no-string-style-prop
```

**Presets**

`dom`
`recommended`
`strict`

## Rule Details

## Common Violations

### Invalid

```tsx
import React from "react";

function MyComponent() {
  return <div style="color: red;" />;
}
```

### Valid

```tsx
import React from "react";

function MyComponent() {
  return <div style={{ color: "red" }} />;
}
```
