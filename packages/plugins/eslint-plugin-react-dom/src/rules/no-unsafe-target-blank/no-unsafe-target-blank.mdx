## Rule Details

## Common Violations

### Invalid

```tsx
import React from "react";

function MyComponent() {
  return (
    <a href="https://eslint-react.xyz" target="_blank">
      Example
    </a>
  );
}
```

### Valid

```tsx
import React from "react";

function MyComponent() {
  return (
    <a href="https://eslint-react.xyz" target="_blank" rel="noreferrer noopener">
      Example
    </a>
  );
}
```
