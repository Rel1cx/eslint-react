## Rule Details

Enforces that the context has a valid component name with the suffix `Context`.

## Common Violations

### Invalid

```tsx
const theme = createContext("");
```

```tsx
const Theme = createContext("");
```

```tsx
const themecontext = createContext("");
```

```tsx
const themeContext = createContext("");
```

### Valid

```tsx
const ThemeContext = createContext("");
```
