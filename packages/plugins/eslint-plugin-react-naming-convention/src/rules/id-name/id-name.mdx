## Rule Details

## Common Violations

### Invalid

```tsx
const value = useId();
//    ^^^ A variable assigned from `useId()` must be named 'id' or end with 'Id'.
```

```tsx
const unique = useId();
//    ^^^ A variable assigned from `useId()` must be named 'id' or end with 'Id'.
```

```tsx
const foo = useId();
//    ^ A variable assigned from `useId()` must be named 'id' or end with 'Id'.
```

### Valid

```tsx
const id = useId();
```

```tsx
const inputId = useId();
```

```tsx
const dialogTitleId = useId();
```
