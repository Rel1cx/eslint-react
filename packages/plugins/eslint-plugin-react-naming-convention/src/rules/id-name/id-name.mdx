## Description

Enforces identifier names assigned from `useId` calls to be either `id` or end with `Id`.

## Examples

### Failing

```tsx
const value = useId();
//    ^^^ A variable assigned from `useId()` must be named 'id' or end with 'Id'.
```

```tsx
const unique = useId();
//    ^^^ A variable assigned from `useId()` must be named 'id' or end with 'Id'.
```

```tsx
const foo = useId();
//    ^ A variable assigned from `useId()` must be named 'id' or end with 'Id'.
```

### Passing

```tsx
const id = useId();
```

```tsx
const inputId = useId();
```

```tsx
const dialogTitleId = useId();
```
