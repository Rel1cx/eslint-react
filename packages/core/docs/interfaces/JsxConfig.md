# Interface: JsxConfig

Resolved JSX configuration derived from compiler options and / or pragma
annotations found in the source file.

## Properties

| Property                                                       | Type      | Description                                           |
| -------------------------------------------------------------- | --------- | ----------------------------------------------------- |
| <a id="property-jsx"></a> `jsx?`                               | `JsxEmit` | The JSX emit mode.                                    |
| <a id="property-jsxfactory"></a> `jsxFactory?`                 | `string`  | The JSX factory function (ex: `React.createElement`). |
| <a id="property-jsxfragmentfactory"></a> `jsxFragmentFactory?` | `string`  | The JSX fragment factory (ex: `React.Fragment`).      |
| <a id="property-jsximportsource"></a> `jsxImportSource?`       | `string`  | The JSX import source (ex: `react`).                  |
