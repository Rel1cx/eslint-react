# no-script-url

## Rule category

Security.

## What it does

Prevents usage of `javascript:` URLs as the value of component props.

## Why is this bad?

Using `javascript:` URLs is harmful to code maintainability, readability, and application security.

## Examples

### Failing

```tsx twoslash
import React from 'react'

function Example() {
  return <a href="javascript:alert('Hello, world!')">Click me</a>
}
```

### Passing

```tsx twoslash
import React from 'react'

function Example() {
  return <a href="/some-page">Click me</a>
}
```
