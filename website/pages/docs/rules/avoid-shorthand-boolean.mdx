## What it does

Enforces the use of explicit boolean values for boolean attributes.

## Examples

### Failing

```tsx
import React from "react";

function Example() {
  return <button disabled />;
  //             ^^^^^^^^
  //             - Avoid using shorthand syntax for 'disabled' attribute.
}
```

### Passing

```tsx
import React from "react";

function Example() {
  return <button disabled={true} />;
}
```
